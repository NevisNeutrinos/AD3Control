# AD2 Test

## Prerequisites

Before you begin, ensure you have met the following requirements:

- Waveforms SDK (Digilent)
- cmake

## Installation

To install the project, follow these steps:

1. make build directory `mkdir build`
2. run cmake on the build directory `cmake ..`
3. run make on the build directory `make`

## Usage

Simply run the executable in the build directory `ad2_test`.
